## Project Overview

ccctx is a Go CLI tool that manages and runs Claude Code with different API contexts. It provides both direct and interactive context selection via a TOML configuration file.

## Quick Reference

- **Tech Stack:** Go 1.23.3, Cobra v1.9.1, Viper v1.20.1, tview
- **Entry Point:** `main.go`
- **Architecture Pattern:** Command Pattern (Cobra CLI)
- **Database:** None
- **Deployment:** Binary distribution via `go install`

## Getting Started

### Prerequisites

- Go 1.23.3 or later

### Setup

```bash
git clone https://github.com/dsdashun/ccctx.git
cd ccctx
make build
```

### Run Locally

```bash
./ccctx list
./ccctx run
./ccctx run <context-name> -- <claude-args>
```

### Run Tests

```bash
make test
```
